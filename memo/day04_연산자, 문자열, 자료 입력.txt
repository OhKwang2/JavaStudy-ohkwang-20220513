@연산자

	#연산자란?
	- 값 : 항
	- 연산자 : 해당 값들을 연산하기 위한 기호

	#단항, 이항, 삼항 연산자, 대입연산자, 부호연산자, 산술연산자
	- 지난시간에 설명하여 내용 작성 생략

	#증감 연산자
	- ++ / -- 앞에 붙을때와 뒤에 붙을 때 구분하여 이해 필요.
	- ++?? : 선진가, ??++ : 후진가
	
	#관계 연산자
	- < , >, >=, <=, ==, !=
	- 결과는 true or false

	#복합 대입 연산자
	- +=, -=, *=, /=, %=
	
	#조건 연산자
	- 조건식 ? 결과1 : 결과2
	- 조건식이 참이면 결과1을 반환, 거짓이면 결과2를 반환
	- 조건식의 결과값은 자료형이 다를 수 없음.(단 업캐스팅이 될 수는 있음)

	#spring 단축키
	ctrl + shift + l : 단축키 리스트가 나타남.

@문자열

	#문자열이란?
	- 여러 문자를 하나로 묶은것
	- String class
	- 문자열은 기본 자료형에 속하지 않는다.
	- API에서 제공하는 String class를 이용

	#문자열 합치기
	- 문자와 문자열은 "+" 연산자로 합침.
