@조건문

	#if, else
	- if : 가정적 조건을 타나냄, (만약) ~면
	- 조건이 참일때만 실행
	- 실행문이 1개이면 {} 생략 가능.
	- if없이 else만 사용 불가 (if는 혼자 사용 가능)

	# else if
	- 하나의 명령일 때 중괄호 생략가능
	※3항 연산자는 결과값만 출력할 뿐 명령을 수행하지 않음.


	#SwitchCase
	- 조건에 입력한 값(조건에 입력하는 값을 연산도 가능함)에 해당하는 case를 찾아 거기서부터 다음 전체 case들을 실행
	- 단, break;를 입력할 경우 해당 case를 찾아 break까지 case들을 실행
	- 입력한 값의 자료형과 case의 자료형은 같아야 함.
	- 중괄호 구분이 없어서 {}내의 전체 값을 실행
	- 값이 없는 경우를 위해 default를 사용 (else와 같은 역할)
	- 단, default를 사용할 경우 default값까지 같이 출력되므로 위에 break;
